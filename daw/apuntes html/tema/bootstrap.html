bootstrap

Caracteristicas

  • bootstrap consiste en trabajar con cajas contenedores

    • la caja contenedora tiene un margen en ambos laterales
    • container-fluid, la caja contenedora ocupa todo el elemento
  • los contenedores se dividen en filas y en 12 columnas
  • el contenedor se puede comportar de manera responsive, es decir, cambiar los que se muestra en una columna para que se vea mejor segun la resolucion donde se mire
  • notacion que se introduce en class para establecer magen, pading, o centrar elementos
  • alineamientos
